Detailed Description

template<template< typename... > typename Visitor, typename Root>
struct arg_router::utility::tree_type_recursor_collector< Visitor, Root >

Similar in principle to tree_type_recursor(), but allows for a type to be built up from the visitor as it recurses.

Visitor must be a template template type like below:

template <typename Current, typename... Parents>
struct my_fn_t {
using type = // Some result type of the visitation
};
