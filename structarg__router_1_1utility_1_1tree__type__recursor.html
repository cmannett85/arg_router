Detailed Description

template<template< typename... > typename Visitor, typename Root>
struct arg_router::utility::tree_type_recursor< Visitor, Root >

Similar in principle to tree_recursor(), but allows for a type to built up from the visitor as it recurses.

Fn must be a template template type like below:

template <typename Current, typename... Parents>
struct my_fn_t {
using type = // Some result type of the visitation
};

my_fn_t<...>::type of all the nodes and policies is concatenated into a std::tuple available in the type member.
