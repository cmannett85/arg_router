Detailed Description

template<typename... AliasedPolicies>
class arg_router::policy::alias_t< AliasedPolicies >

Allows the 'aliasing' of arguments, i.e. a single argument will set multiple others.

Note
An aliased argument cannot be routed, it's aliased arguments are set instead

Duplicates any value tokens as aliases of other nodes.

The owning type must have minimum_count() and maximum_count() methods that return the same value, or a count() method(), to use an alias.

  • If the count is zero then it is flag-like so the aliased names are just inserted into tokens after the current token
  • If the count is greater than zero then it is argument-like and the aliased names are inserted, each followed by count tokens (i.e. the value), after the current count tokens
