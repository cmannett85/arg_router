Detailed Description

template<typename... Policies>
class arg_router::flag< Policies >

Represents a flag in the command line.

A flag is a boolean indicator, it has no value assigned on the command line, its presence represents a positive boolean value.

Flags with shortnames can be concatenated or 'collapsed' on the command line, e.g.

foo -a -b -c
foo -abc
Template Parameters
PoliciesPack of policies that define its behaviour
