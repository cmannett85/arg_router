template<typename... ParentIndexTypes>
struct arg_router::policy::validation::parent_types< ParentIndexTypes >

A rule condition that checks one of the parent index and type pairs given in ParentIndexTypes matches Parents.

ParentIndexTypes can have multiple entries against the same index, as long as one of the types is a match then the check will pass.
