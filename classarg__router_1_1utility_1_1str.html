Detailed Description

template<detail::compile_time_string_storage S = detail::compile_time_string_storage<0>{}>
class arg_router::utility::str< S >

Compile-time string.

This type is only used when C++20 and greater is supported, and when AR_DISABLE_CPP20_STRINGS is false.

Like the C++17 equivalent S_(..) macro, this can accept a string literal directly.

Note
Unlike the S_(..) macro, this cannot use a std::string_view as an input type, use std::array<char, N> or std::span<char, N> instead
