Detailed Description

template<typename Validator>
class arg_router::parsing::pre_parse_data< Validator, false >

Without parse_target reference specialisation.

The Validator instance is called just before the args list is updated by the pre_parse(..) method, and allows the caller to run a custom verification the on the method's node and parents arguments. Validator must be equivalent to:

struct my_validator {
template <typename Node, typename... Parents>
bool operator()(const Node&, const Parents&...);
};

If the validator returns true then the result is kept.
