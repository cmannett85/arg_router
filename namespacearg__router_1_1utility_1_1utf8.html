Detailed Description

Namespace for UTF-8 encoded string functions.

Note
These are not true UTF-8 algorithms as they make the assumption that 1 code point is equal to one user-perceived character (a.k.a. grapheme cluster) - this is not always true! For true UTF-8 support, you must link against ICU
