template<typename Visitor , typename Root >
constexpr void arg_router::utility::tree_recursor ( )
constexpr

Depth-first recurse through the parse tree, calling Visitor on every tree_node and policy.

Fn must be object with the same method signatue as below:

struct my_fn {
template <typename Current, typename... Parents>
constexpr static void fn()
{
...
}
};

Where Current is the current type from the tree, from the root down to policy level, and Parents is a pack of ancestors in increasing generation from Current. The last in the pack is always the root unless Current is itself the root.

template<typename Visitor , typename SkipFn , typename Root >
constexpr void arg_router::utility::tree_recursor ( )
constexpr

Specialisation of tree_recursor() where a skip function can be used to ignore subtrees.

SkipFn must be object with the same method signatue as below:

struct my_skip_fn {
template <typename Current, typename... Parents>
constexpr static bool fn()
{
...
return condition_met;
}
};

If the return value is true, then the Current node and any subtree below it are skipped i.e. Fn is not called.
