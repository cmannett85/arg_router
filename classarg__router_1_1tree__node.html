inlineconstexpr

Finds the child matching token (if present) and calls visitor with a reference to it.

visitor needs to be equivalent to:

[](auto i, const auto& child) { ... }

i is the index of child in the owner's child list.

This calls the match(..) const method on each child until it returns true.
